1. **Борьба за сахар** — дано поле `N×M` с запрещёнными клетками, нужно разместить как можно больше прямоугольных конфет `2×1` (можно поворачивать) так, чтобы они не пересекались и не покрывали зачёркнутые клетки; требуется вывести максимальное число покрытых клеток. Сложность — сводится к максимальному паросочетанию в двудольном графе клеток (шахматная раскраска + рёбра между соседями), при `N,M ≤ 100` важно сделать эффективно по памяти/времени.

2. **Шёпот далёких звёзд** — дан ориентированный граф с пропускными способностями, где всегда источник `1`, а сток — `N`; нужно найти максимальный поток (максимум “слёз”), который может получить Чёрная Дыра. Сложность — классический max-flow при `N,M ≤ 10^4`, важно подобрать алгоритм (Диниц/Эдмондс–Карп) и не провалиться по времени на плотных графах.

3. **А можно я с тобой?** — дан связный неориентированный взвешенный граф, нужно найти максимальное паросочетание по суммарному весу (выбрать набор непересекающихся рёбер с максимальной суммой длин). Сложность — для общих графов это нетривиальная задача (алгоритмы уровня блоссома/взвешенного matching), плюс большие ограничения (`N до 10^5`, `M до 10^6`) требуют очень аккуратной реализации и оптимизаций.

4. **Я не продаюсь, но...** — задача стабильных пар: есть `N` котят и `N` хозяев, у обеих сторон полные упорядоченные списки предпочтений; нужно получить стабильное соответствие, причём предпочтения котят важнее (они “делают предложения”). Сложность — реализация алгоритма Гейла–Шепли на `N ≤ 1500` с быстрым сравнением рангов (таблица позиций) и корректным выводом пар, отсортированных по номеру котёнка.

5. **Летняя тоска Светланы** — транспортная сеть с пропускными способностями `c` и стоимостью `p` на ребре; нужно вывести максимальный поток из `s` в `t` и минимальную стоимость среди всех потоков максимального значения (min-cost max-flow). Сложность — комбинированная оптимизация: сначала “добиваем” максимум по пропускной способности, параллельно минимизируя стоимость; обычно решают min-cost max-flow (потенциалы + Дейкстра/SPFA) при `N ≤ 2000` и плотном графе.

6. **Highway of a lifetime** — дан граф, для которого гарантировано существование гамильтонова цикла (по теоремам Оре/Дирака), нужно восстановить и вывести такой цикл. Сложность — не поиск “в общем виде” (он NP-трудный), а аккуратная конструкция по известному свойству графа и требуемому порядку обработки вершин, причём `N` до `10^5`, `M` до `10^6`, поэтому нужна линейно-логарифмическая реализация без тяжёлых проверок.
